# Training-Modules

This repository contains the weekly training modules provided by my college as part of the Summer Training Program.

## Overview

The repository is organized week-wise and includes the questions, exercises, and tasks assigned during the training period. It serves as a record of the learning material covered throughout the program.

## Contents

* Weekly training modules
* Practice questions
* Assignments and exercises
* Learning resources (if provided)

## Purpose

* Track progress throughout the summer training.
* Maintain a structured collection of weekly assignments.
* Practice and strengthen technical concepts covered during the program.

## Notes

The questions and materials in this repository are provided as part of the college summer training curriculum and are intended for learning and practice purposes.
